    # CfTextfield

    • 示例
    <template>
    <div class="comp-container">
        <cf-text-field v-model="inputText" :checkFn="handleCheckFn" required label="Input your name:"></cf-text-field>
        {{ inputText }}
        <cf-text-field v-model="inputText2" :disabled="true" :checkFn="handleCheckFn" label="Input your age:"></cf-text-field>
        {{ inputText2 }}
    </div>
</template>
<script>
export default {
    name: "CalendarDemo",
    data() {
        return {
            inputText: 'Abc',
            inputText2: '30'
        }
    },
    methods:{
        handleCheckFn(text){
            if(text.length > 3) throw new Error('长度不能大于 3')
        },
    }
}
</script>
    • 参数列表
    名称 类型 可选值 默认值 描述
    value string - - 字段值
    label string - - 字段名
    required boolean - false 是否必输
    checkFn function - - 检验回调
    disabled boolean - false 是否禁用
